# Java面试题

# Java基础

# Java八大基础类型

  • 数字型：字节类型(byte)、短整型(short)、长整型(long)、整型(int)、单精度浮点数(float)、双精度浮点数(double)
  • 字符型：字符类型(varchar)
  • 布尔型：布尔类型(boolean)

# Java三大特性

  • 封装：使用private关键字，让对象私有，防止无关的程序去访问使用。
  • 继承：继承某个类，使子类可以使用父类的属性或方法。
  • 多态：同一个行为，不同的子类具有不同的表现形式。

# 重载和重写的区别

  • 重载：发生在同一个类中，函数名必须一样，参数类型，参数个数，参数顺序，返回值类型，修饰符可以不一样。
  • 重写：发生在父子类，函数名、参数、返回值必须一样，访问修饰符必须大于等于父类，异常要小于等于父类，父类方法是private不能重写。

# public、protected、（default）不写、private修饰符的作用范围

  • public：同类、同包子类、非同包子类、非同包类都可以使用。
  • protected：同类、同包子类、非同包子类都可以使用，非同包类不能。
  • (default)不写：同类、同包子类都可以使用，非同包子类、非同包类不能。
  • private：只有同类可以。

# ==和equals的区别

  • ==：基础类型比较的是值，引用类型比较的是地址。
  • equals：没有重写比较的是地址是否相等，重写比较的是内容是否相等。

# hashcode()值相同，equals就一定为true？

不一定，重写hashcode或者属性相同都有可能导致hashcode值相等，但equals确实是false； 即使两个对象的hashcode值不相等，它们的内容也可能相同，这个对重写过的equals()来说就为true； 但是equals()为值，那么hashcode()就一定相同。
